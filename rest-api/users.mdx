Manage user accounts, track their usage, and associate them with plans using the Limitly REST API.

## List Users

Get all users for your account:

```bash
curl -X GET \
  -H "Authorization: Bearer lk_1234567890abcdef1234567890abcdef" \
  https://api.limitly.dev/v1/users
```

### Response

```json
{
  "data": [
    {
      "user_id": "user_123",
      "name": "John Doe",
      "email": "john@example.com",
      "plan_id": "plan_456",
      "created_at": "2024-01-01T00:00:00Z",
      "updated_at": "2024-01-01T00:00:00Z",
      "metadata": {
        "company": "Acme Corp",
        "role": "developer"
      }
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 1
  }
}
```

## Create User

Create a new user account:

```bash
curl -X POST \
  -H "Authorization: Bearer lk_1234567890abcdef1234567890abcdef" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Doe",
    "email": "john@example.com",
    "plan_id": "plan_123"
  }' \
  https://api.limitly.dev/v1/users
```

### Request Parameters

<ParamField body="name" type="string" required>
  The name of the user
</ParamField>

<ParamField body="email" type="string" required>
  The email address of the user
</ParamField>

<ParamField body="plan_id" type="string">
  The ID of the plan to associate with this user
</ParamField>

<ParamField body="metadata" type="object">
  Additional metadata for the user
</ParamField>

### Response

```json
{
  "data": {
    "user_id": "user_123",
    "name": "John Doe",
    "email": "john@example.com",
    "plan_id": "plan_456",
    "created_at": "2024-01-01T00:00:00Z",
    "updated_at": "2024-01-01T00:00:00Z",
    "metadata": {}
  }
}
```

## Update User

Update an existing user:

```bash
curl -X PUT \
  -H "Authorization: Bearer lk_1234567890abcdef1234567890abcdef" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Smith",
    "plan_id": "plan_789",
    "metadata": {
      "company": "Acme Corp",
      "role": "senior_developer"
    }
  }' \
  https://api.limitly.dev/v1/users/user_123
```

### Request Parameters

<ParamField body="name" type="string">
  New name for the user
</ParamField>

<ParamField body="email" type="string">
  Updated email address
</ParamField>

<ParamField body="plan_id" type="string">
  Updated plan ID to associate with the user
</ParamField>

<ParamField body="metadata" type="object">
  Updated metadata for the user
</ParamField>

## Delete User

Delete a user account (soft delete):

```bash
curl -X DELETE \
  -H "Authorization: Bearer lk_1234567890abcdef1234567890abcdef" \
  https://api.limitly.dev/v1/users/user_123
```

### Response

```json
{
  "data": {
    "message": "User deleted successfully"
  }
}
```

## Error Responses

### User Not Found

```json
{
  "error": "User not found",
  "code": "USER_NOT_FOUND"
}
```

### Validation Error

```json
{
  "error": "Validation failed",
  "code": "VALIDATION_ERROR",
  "details": {
    "field": "email",
    "message": "Email is required"
  }
}
```

### Duplicate Email

```json
{
  "error": "Email already exists",
  "code": "DUPLICATE_EMAIL"
}
```
