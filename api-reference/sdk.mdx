# SDK Reference

Complete reference for the Limitly JavaScript SDK.

## Limitly Class

The main SDK class that provides access to all Limitly functionality.

### Constructor

```typescript
new Limitly(config: LimitlyConfig)
```

#### Configuration Options

<ParamField body="apiKey" type="string" required>
  Your Limitly API key
</ParamField>

<ParamField body="baseUrl" type="string">
  The base URL for the Limitly API. Defaults to `https://api.limitly.com`.
</ParamField>

<ParamField body="timeout" type="number">
  Request timeout in milliseconds. Defaults to 30000 (30 seconds).
</ParamField>

### Properties

<ParamField body="apiKeys" type="ApiKeysModule">
  Module for managing API keys
</ParamField>

<ParamField body="plans" type="PlansModule">
  Module for managing plans
</ParamField>

<ParamField body="users" type="UsersModule">
  Module for managing users
</ParamField>

<ParamField body="validation" type="ValidationModule">
  Module for validating requests
</ParamField>

### Methods

<ParamField body="getClient()" type="HttpClient">
  Gets the internal HTTP client. Useful for debugging and testing.
</ParamField>

## Error Handling

The SDK throws `LimitlyError` for API errors:

```typescript
try {
  const result = await limitly.validation.validate('invalid_key', '/api/test', 'GET');
} catch (error) {
  if (error instanceof LimitlyError) {
    console.log('API Error:', error.message);
    console.log('Error Code:', error.code);
    console.log('Status Code:', error.statusCode);
  }
}
```
